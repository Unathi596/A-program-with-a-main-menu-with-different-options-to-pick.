# A-program-with-a-main-menu-with-different-options-to-pick.
## Option A: Number Patterns ##
- First the user must input the initial value (Value must be greater than or equals to zero)
- User also inputs the number of terms needed to generate the sequence (Number of terms must be greater than zero)
### The Pattern goes like this ###
- The user first provides the initial value (For Example "1")
- The user provides us with the number of terms needed (Let's say the user inputs "4")
- The sequence forms when we square the Values then we add 2 to the original value. (We continue to do this according to the number of terms in case 4 terms are needed, so we do this 4 times)
- Like this [1 squared = 1, 1+2 = 3]  [3 squared = 9, 3 + 2 = 5] [5 squared = 25, 5+2 = 7] [7 squared = 49] 
- Therefore we only display the squared values (1,9,25,49)

## Option B: Prime numbers ##
-This is program that determines whetther a number is prime or not.
- The factors of that number is also displayed.
- The user just inserts the number.
- Then the factors of that number will be displayed.
- Then the compiler determines whether the number is prime or not. (Remembers prime numbers has only two factors, 1 and the number itself)

## Option C: Non-Vowel Characters in a string ##
- This is a program that determines Non-Vowel Characters in a string.
- User just inputs any number of characters.
- The program will display only the non-vowel characters in a string.

## Option X: Exit Program ##
- This will just exit the program if the user inputs the character 'x'

## Concepts used ##
- Loops (While, For, For each, Do-While)
- Error handling 
- Switch Cases
- Clearning the buffer (cin.ignore)
